Stream Lit

-Python library to build web apps using python only
-No need of HTML,CSS or JavaScript
-Used for : ML Projects ,Data apps & Dashboards

------------------------How TO Install Stream Lit-----------------
pip install streamlit

-> streamlit run app.py

Basic Concepts
1.Script Based
2.Auto Refresh
3.Rerun concepts

1.Script Based:normal python file -> becomes web application
2.Auto Refresh:Save File-> app updates automatically
3.Rerun concepts: Every user action (buttons,input)-> entire code runs again , used for updating purpose,auto updates whenever user click a button

Functions under stream list
Imp functions for stream lit:
1.  st.title()-> Title
2.  st.write()-> Display anything
3.  st.text_input()->Input box(entry)
4.  st.button()->Button
5.  st.selectbox()->Dropdown(option menu)
